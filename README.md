# Instructions pour utiliser le package curobo_ros

## Utilisisation du service de calcul de cinematique directe de cuRobo :

### Lancer le service curobo_fk
Pour lancer le service FK, exécutez la commande suivante :

```bash
ros2 run curobo_ros curobo_fk
```

### Calcul de cinematique directe a partir d'une trajectoire calculee par cuRobo :
Veuillez exécutez la commande suivante :

```bash
ros2 run curobo_ros curobo_test
```
